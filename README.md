# 60 Days DSA Challenge - #GeekStreak60
> 60 days. 60 problems. One goal - consistency.

This repository documents my journey of completing the **60 Days Problem of the Day (POTD)** challenge by GeeksforGeeks in collaboration with NPCI.

---

## Overview

I maintained a continuous 60-day streak by solving one DSA problem daily without missing a single day.
This repository is not just a collection of solutions —
it reflects consistency, structured thinking, and disciplined problem-solving.

---

## What This Repository Demonstrates

*  Consistency - solving problems daily for 60 days
*  Pattern Recognition - Sliding Window, Prefix Sum, Stack, etc.
*  Problem-Solving Approach - Understand → Plan → Implement
*  Clean and readable code
